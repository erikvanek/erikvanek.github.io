# Gemini Agent Context: The Second Brain (PKM)

## 1. System Philosophy: "The Garden of Knowledge"
This vault is a Personal Knowledge Management (PKM) system built on **PARA** (Projects, Areas, Resources, Archives) and **BASB** principles.

**Core Goal:** "Start projects that are already 80 percent done" by leveraging gathered notes and unusual connections.

## 2. Mandatory Principles for AI Assistance
1.  **Strict English-Only:** All outputs (note names, content, tags, summaries) MUST be in English. If the input is in Czech, translate it during processing.
2.  **Human-in-the-Loop:** Propose changes or structures, but always wait for user confirmation for permanent filing.
3.  **PARA Alignment:** Always categorize knowledge into:
    *   `1 - 🚀 Projects`: Active, time-bound tasks (Private Submodule).
    *   `2 - 🌱 Areas`: Long-term responsibilities (Design, Product, Personal).
    *   `3 - 📚 Resources`: Interests/Library for future reference.
    *   `4 - 🏛️ Archives`: Completed/Inactive items (Private Submodule).
4.  **Store RAW, Enrich Existing:** Keep original notes in `04 - 💽 RAW` for provenance. Use new information to enrich "knowledge nuggets" in `10 - 🧠 Knowledge`.

## 3. Specialized Domain Knowledge
When assisting with projects, leverage these existing frameworks found in `10 - 🧠 Knowledge/3 - 📚 Resources`:
*   **Design:** Service Design, UX Metrics (HEART, NPS), Behavioral Design (Nudges, Hook model), Design Research.
*   **Product:** Outcome-over-Output, Product Market Fit, Product Strategy, Agile (Vibe coding/Design Engineering).
*   **Storytelling/Persuasion:** Hero's Journey, FAB Framework (Feelings, Actions, Beliefs), Aristotle's Triad (Ethos/Pathos/Logos).

## 4. Navigating the Lifecycle
*   **Processing:** Check `99 - 📄 To process` for current synthesis work.
*   **Metadata:** Use existing templates in `01 - 🤖 Second brain/Templates` for consistency.
*   **Routines:** Refer to `01 - 🤖 Second brain/Routines/` for specific workflows (Inbox processing, Retrieval, Rebalancing).

## 5. Security & Privacy
*   Be aware that `10 - 🧠 Knowledge/1 - 🚀 Projects` and `10 - 🧠 Knowledge/4 - 🏛️ Archives` are private submodules. 
*   Never leak private submodule content into the public main repository.
