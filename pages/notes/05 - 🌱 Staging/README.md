# Staging

Incubation layer for content that's interesting but not yet validated as worth keeping in Knowledge.

**Graduation heuristic**: After 2–4 weeks, ask *"Would I actively use this in a project?"* If yes → migrate to `10 - 🧠 Knowledge`. If no → archive or discard.

Review during Friday weekly check-in. Anything older than ~4 weeks should have a decision.

See [[Revival-Strategy]] for full context.
