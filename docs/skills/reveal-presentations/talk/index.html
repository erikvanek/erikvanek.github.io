Content Guidelines

  • Hook early ‚Äì why should audience care?
  • Keep intro short ‚Äì ~2 slides max about yourself
  • Visual variety ‚Äì alternate text slides with full-bleed images/videos
  • Check in periodically ‚Äì "Co se v√°m hon√≠ hlavou?"
  • Concrete examples ‚Äì name companies, products, projects
  • End with resources ‚Äì give audience places to go deeper
  • Leave Q&A time ‚Äì don't cram content to the last minute

Pacing

For a 45-minute talk:

  • 5 min: intro + context
  • 30 min: core content (~15-20 slides)
  • 5 min: wrap-up + resources
  • 5 min: Q&A

  • Use r-fit-text for impact statements and transitions
  • Full-bleed backgrounds for story moments
  • Consistent image styling across examples
  • Video backgrounds for memorable closing
